Om detta projekt

Det här är mitt projektarbete i kursen JavaScript. Målet är att skapa ett interaktivt quiz som visar att jag kan använda JavaScript för att göra webbsidor interaktiva.

Vad projektet innehåller

Jag har använt:

  • DOM-manipulation för att skapa och ändra element
  • Events för att hantera klick och tangentbord
  • Arrays och objekt för att lagra frågor och svar
  • Validering för att kolla att användaren svarat på allt
  • Funktioner för att organisera koden

Sidorna

Webbplatsen har fyra sidor:

  • Startsida - Översikt över quiz
  • JavaScript Quiz - 30 frågor
  • Om - Den här sidan
  • Kontakt - Kontaktformulär

Framtida planer

I framtiden kan jag lägga till fler quiz om HTML & CSS, Introduktion till IT och eventuellt framtida kurser. Bland mer långsiktiga förändringar så kan jag lägga till en tidsbegräsning, en poängräknare och en databas för att spara resultat.
